# スクールエンジン 年度初め処理

スクールエンジンの年度初めに行う一連の設定作業の手順書です。

## 1. 転任職員の追加
- **メニュー**: システム管理 → 教職員マスタ
- **操作**: 転任してきた職員を追加します。
- **ポイント**: 登録の際、職員コードとして **1xxxxxx** を確認・入力してください。

## 2. 授業日の設定
- **メニュー**: 教務 → 授業日設定
- **操作**: 当年度の授業日カレンダーを設定します。

## 3. 評価時期の設定
- **メニュー**: 教務 → マスタ設定 → 評価時期マスタ
- **操作**: 年間計画に基づき、評価時期を設定します。
- **ポイント**:
    - **仮評定1**: 1学期期末終了日に設定します。
    - **変更タイミング**: その後、2次募集などの進捗に合わせて、2学期期末の日付に変更します。

## 4. 新入生登録
- **メニュー**: 教務 → 生徒情報設定 → 学籍データ読み込み
- **操作**: 新入生のデータを設定します。
- **手順**: 教務主任より提供される「入試データ」と「新1年生の名簿」をもとに、CSV形式で一括登録を行います。
    - ※詳細は別途記載予定。

## 5. 授業担当者の設定
- **メニュー**: 教務 → 講座の作成・受講処理 → 講座の詳細設定
- **操作**: 各講座（約250講座）の担当教諭（複数教員を含む）を設定します。
- **ポイント**: 
    - 授業データは昨年度のものがコピーされているはずですが、担当者は改めてすべて入力する必要があります。
    - **体育**: 設定が複雑なため、最適な設定方法については教務主任等に相談することをお勧めします（※一括設定機能はおそらくありません）。

## 6. 生徒別履修登録（一括）
- **メニュー**: 教務 → 履修登録・教科書 → 履修パターンによる一括登録
- **操作**: 年度と学科を選択し、クラス別で履修授業を登録します。
- **ポイント**: 
    - 履修パターンは「令和4年度〇〇」などの適切な最新パターンを選択してください。
    - 2年生以降の1〜5組などは、30単位中28単位ほどがこの段階で登録されます（選択科目が未登録のため）。
    - 履修パターンを選択授業まで含めて作成しておけば個別登録の手間が省けます。今後の運用を考慮すると、作成を検討してください。

## 7. 生徒別履修登録（選択科目・個別修正）
- **メニュー**: 教務 → 生徒別履修登録
- **操作**: 一括登録で漏れた選択科目の追加や、個別の修正を行います。

## 8. 履修科目講座への振り分け
- **メニュー**: 教務 → 講座作成受講処理 → 履修科目講座への振り分け
- **操作**: 全生徒の履修登録（30〜31単位分）を、特定の担当者（講座）に紐付けます。
- **例**: 「現代の国語」を1年1組、1年2組……と全クラス・全講座分振り分けを行ってください。
