# Manual Hub の使い方

Manual Hubは、あなたの知識を構造化して管理するためのツールです。Antigravityに依頼してマニュアルを追加・編集することができます。

## 操作手順
1. タイトル、ジャンル、タグを設定してマニュアルを作成します。
2. 保存されたマニュアルは、ジャンルやタグ、キーワードで検索可能です。
3. Antigravityに「[タイトル]の内容を修正して」と指示することで、内容を即座に更新できます。

## ポイント
- **シンプルさ**: 複雑な設定なしですぐに使えます。
- **検索性**: 大量のドキュメントから目的の情報を素早く見つけ出せます。
