# 2026-04-28(火) R80メソッド わかりやすい解説版

## R80メソッドとは

**80字で、自分の考えを論理的に書く方法**

---

## 作文の基本形：4つのパーツ

```
【結論】+ 【接続詞】+ 【感想A】+ 【感想B】
```

### 1. 結論（テーマへの答え）
テーマについて、自分の主張や説明を簡潔に述べる

### 2. 接続詞（流れをつなぐ）
「しかし」「なぜなら」「だから」「また」「いっぽう」「つまり」など

### 3. 感想A（最も伝えたいこと）
5W1Hと五感から集めたメモの中で、最も重要なポイント

### 4. 感想B（感想Aに付随する補足）
感想Aを支える、関連する別の視点やポイント

---

## 実例で見る構造

### テーマ：「晩御飯」

| パーツ | 内容 | 作文での表現 |
| ---- | ---- | ---- |
| **結論** | 何を食べたのか | 母親が作ってくれた大好物のハンバーグ |
| **接続詞** | つなぎ役 | しかし |
| **感想A**（重要） | 何が起きたか | 食べ過ぎて体重が1kg増えた |
| **感想B**（補足） | その影響 | 明日の身体測定に影響する |

### 完成文

「私の昨日の晩御飯は母親が作ってくれた大好物のハンバーグです。**しかし**、あまりにもおいしかったため食べ過ぎて、体重が1kgも増えてしまったので、明日の身体測定には大きな影響を与えることになってしまいました。」

---

## 感想AとBを見つける方法：3ステップ

### ステップ1：とにかくメモしまくる

「晩御飯」なら、以下を全部メモする：

**5W1H**
- When（いつ）→ 昨日の晩
- Where（どこで）→ 家の食卓で
- Who（誰が）→ 母親が
- What（何を）→ ハンバーグを
- Why（なぜ）→ 大好物だから
- How（どのように）→ おいしそうに作ってくれた

**五感**
- 視覚：こんがり焼けた色
- 聴覚：ジュージューという音
- 嗅覚：香ばしい匂い
- 味覚：ジューシーでおいしい
- 触覚：温かい

### ステップ2：チェックマークを付ける

メモの中から「一番読み手に伝えたいこと」にチェック✓

例：「食べ過ぎて体重が1kg増えた」← ここが最も伝えたいポイント

→ これが **感想A**

### ステップ3：感想Bを追加

感想Aに関連する別のポイントを選ぶ

例：「明日の身体測定に影響する」← 感想Aの結果

→ これが **感想B**

---

## 作文完成までの流れ

```
テーマ決定
    ↓
5W1Hと五感でメモしまくる（10～20個）
    ↓
一番伝えたいポイントを選ぶ（感想A）
    ↓
関連するポイントを選ぶ（感想B）
    ↓
結論＋接続詞＋感想A＋感想B で80～100字の作文が完成
```

---

## ポイント

- **論理的**：結論→理由→結果で筋道が通る
- **簡潔**：80～100字に収める
- **量産可能**：この型を繰り返すだけで、次々と作文が書ける
