# 2026-04-22(火) 就職進学統計調査について

進路指導部での依頼事項

## 概要
2026年4月22日に事務より依頼された、昨年度の就職者・進学者の行先を提出する業務。

## 対象データ
昨年度の就職者および進学者の進路先情報

## 提出内容

### 就職者に関する対応
- 内定者データを作成する
- 職種と業種のコードを使用して振り分けをする
- 内定者データをしっかり作成しておくと、後々の処理がスムーズになる
- 詳しい作成方法は後日記載予定

### 参照するコード分類

- **日本標準産業分類**（令和５年[2023年]７月改定）
- **厚生労働省編職業分類**（令和４年改定）

### 産業分類コードの読み方

- コードは `311` や `091` のような3桁形式
- **左2桁**を分類番号として使用（例：311→31、091→09）
- 文部科学省の産業別振り分け（A～T）に対応

| 記号 | 産業 |
|------|------|
| A | 農業・林業 |
| B | 漁業 |
| C | 鉱業・採石業・砂利採取業 |
| …  | D～T は後日記載 |

### 職業分類コードの読み方

- コードは `74-11` や `09-99` のような形式
- **厚生労働省編職業分類**と一致
- **前半2桁**が大分類

**厚生労働省編の大分類（例）**

| 記号 | 職業 |
|------|------|
| A | 管理的職業 |
| B | 研究・技術の職 |
| … | 後日記載 |

### 日本標準職業分類（文部科学省・総務省）への変換

厚生労働省編の前半2桁をもとにA～Lへ振り分け：

| 厚生労働省編 前半2桁 | 日本標準職業分類 |
|----------------------|-----------------|
| 1～3 | A：管理的職業従事者 |
| 4～11 | B：専門的・技術的従事者 |
| … | 後日記載 |

## データ構造（予定）

### 列方向（学科別）
- 商業男子
- 商業女子
- 生活文化男子
- 生活文化女子
- 福祉男子
- 福祉女子

### 行方向（業種・職種）
- A～Lのように業種と職種を別シートで作成予定

## 報告先

作成された内定者データをもとに以下への報告が必要：
- ハローワーク
- 総務省
- 教育委員会

何の文章をどのように提出するかは、その都度追記予定
